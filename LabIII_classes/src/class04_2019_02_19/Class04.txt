Contatenação em Java pode ser visto com o sinal de "+" que para string faz a junção das strings.

Exemplo:
    S1 = "abc"
    S2 = "edf"
    S3 = S1 + S2
    // S3 = "abcdef"

O final de "+=" também serve para fazer a concatenação.
